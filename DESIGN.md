# 美团AI · 周末出行规划 Agent — 设计文档

## 概述

面向手机端打造的服务美团生态的行程规划智能体，实现 **手机定位获取 → 周边POI快速检索 → 家庭/好友双模式路线规划 → 跳转美团预约下单** 整套功能。

## 核心模块设计

### 1. 位置获取模块

- **技术方案**: 浏览器 HTML5 Geolocation API
- **精度需求**: `enableHighAccuracy: true`, 8秒超时
- **降级策略**: 定位失败自动使用默认位置（北京朝阳区中心）
- **手动调整**: 支持手动输入地址 + 快捷地址选择（朝阳大悦城、三里屯等）
- **地址→坐标**: 内置热点区域坐标映射表，模拟地理编码

### 2. 周边POI数据检索模块

- **数据规模**: 30+ 模拟POI，覆盖 5 大品类
  - 餐饮（10家）: 火锅、北京菜、西北菜、小龙虾、茶饮等
  - 景点（7家）: 公园、艺术区、主题乐园、商业街区
  - 亲子（6家）: 动物园、科技馆、儿童乐园、室内乐园
  - 休闲娱乐（6家）: 密室逃脱、KTV、电影院、汤泉、轰趴馆
  - 购物（5家）: 购物中心、高端百货、商业街区
- **POI字段**: 名称、品类、坐标、评分、人均消费、营业时间、亲子友好度、美团链接
- **筛选维度**: 距离远近(Haversine)、评分高低、消费预算、品类
- **检索范围**: 默认半径 5-8km

### 3. 智能最优路线规划模块

**算法**: 贪心最近邻 + 多因子评分加权

**评分因子**:
- 模式适配分 (familyFriendly vs trendy)
- 距离惩罚 (越近越好)
- 类别覆盖奖励 (优先未覆盖的品类)
- 时间匹配度 (剩余时间充裕度)
- 避免同类相邻

**家庭模式策略**:
- 偏好品类: 景点 > 亲子 > 餐饮 > 购物 > 休闲娱乐
- 亲子友好度加权 ×2
- 排除密室逃脱、KTV、夜店类
- 出行方式: 驾车
- 默认出发时间: 上午 9:30

**好友模式策略**:
- 偏好品类: 休闲娱乐 > 餐饮 > 购物 > 景点 > 亲子
- 打卡网红加权
- 排除亲子专属场所
- 行程更紧凑
- 默认出发时间: 下午 13:00

**时间线优化**:
- 自动计算站间距离与出行时间
- 标注到达/离开时间
- 15分钟站间缓冲
- 总时长控制在用户指定范围内

### 4. 美团应用联动模块

**跳转策略**:
1. 优先尝试唤起美团APP (`meituan://` URL Scheme)
2. 1.5秒超时后fallback到美团网页版 (`https://i.meituan.com/`)
3. 使用隐藏iframe方案避免页面跳转中断

**模拟下单流程**:
- 一键预订全部: 自动筛选需要预订的POI
- 单个预约: 点击每个POI的"立即预约"按钮
- 订单状态展示: 已预约/待确认
- 美团APP确认: 模拟跳转至美团完成支付

### 5. 智能交互模块

**NLP意图识别**:
- 基于正则表达式 + 关键词词典的轻量级NLP
- 识别维度: 出行模式、人数、时长、预算、偏好
- 支持中英文数字混合表达（"3个人" / "三个人"）

**对话流程**:
1. 用户输入自然语言需求
2. 系统解析意图并确认理解
3. 后台规划路线
4. 输出行程摘要
5. 支持多轮调整（换餐饮、加购物、缩短行程等）

**快捷操作**:
- 首页场景卡片（家庭/好友一键启动）
- 对话中快捷回复按钮
- 调整方案的预设选项

## Planning 策略

```
输入用户需求
    ↓
NLP意图解析 (模式/人数/时长/预算/偏好)
    ↓
POI检索 (范围筛选 → 模式评分 → 综合排序)
    ↓
贪婪路线构建 (最多5-6站，时间约束)
    ↓
时间线优化 (出行方式/时间标注/缓冲)
    ↓
输出完整行程 + 美团可操作链接
```

## 异常处理机制

| 场景 | 处理方式 |
|------|----------|
| 定位失败/拒绝 | 使用默认位置（北京朝阳区），弹Toast提示 |
| 周边POI不足 | 扩大检索半径至8km |
| 时间不足以覆盖所有点位 | 贪婪算法自动裁剪，保证不超时 |
| 美团APP未安装 | 1.5s超时后自动跳转网页版 |
| 分享API不可用 | 降级为剪贴板复制 |
| 用户输入无法解析 | 使用默认值（家庭模式，5小时） |

## 技术选型

- **框架**: 原生 HTML/CSS/JS（零依赖、零构建步骤）
- **定位**: Browser Geolocation API
- **样式**: CSS Variables + Mobile-First 响应式
- **最大宽度**: 480px（手机端适配）
- **字体**: Noto Sans SC (Google Fonts)
- **品牌色**: 美团黄 #FFD100
